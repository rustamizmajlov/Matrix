# Matrix Multiplication (C++)

Измайлов Рустам Ринатович МПБ-301

Учебный проект: перемножение двух матриц A и B с использованием самописной структуры `Matrix`
(хранение элементов в одномерном `vector` и доступ через `operator()`).

## Стек

- C++17 (подойдет и C++14/20)
- Компилятор: g++ / clang++
- Внешние библиотеки: не используются
